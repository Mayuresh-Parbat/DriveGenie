# 🚗 DriveGenie – AI Car Marketplace

DriveGenie is an AI-powered car marketplace that helps users find the best cars using intelligent search and recommendations.

## 🚀 Features (Planned)
- AI-based car search
- Smart recommendations
- Car comparison
- Deal analyzer

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js
- AI: Python
- Database: MongoDB
